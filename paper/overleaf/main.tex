% -------------------- Abstract --------------------
\begin{abstract}
Edge--cloud visual simultaneous localization and mapping (VSLAM) can exploit cloud-side perception and mapping, but it requires transmitting dense visual streams from resource-constrained edge devices, causing substantial bandwidth and energy costs. Conventional image and video codecs mainly optimize pixel-level or perceptual fidelity, which may not preserve the geometric continuity and trackable structures needed by VSLAM. This paper proposes \HVM, a human-inspired image--text visual memory coding framework for VSLAM-oriented edge--cloud visual transmission. Instead of sending dense frames, the edge encoder represents a visual sequence using sparse boundary frames, motion memory, visual novelty anchors, and compact language memory. On the cloud side, an image--text conditioned world-model decoder reconstructs task-usable intermediate views under the transmitted constraints, and Low-Rank Adaptation (LoRA) is used to bias the decoder toward VSLAM-oriented reconstruction. The proposed codec is evaluated using downstream absolute pose error (APE) RMSE, transmission reduction, edge-side motion-estimation cost, and auxiliary LPIPS, SSIM, and FID metrics. Experiments show an average 96.8\% transmission reduction, GEN APE RMSE close to ORI APE RMSE on average, low-cost phase-correlation motion memory, and a reduction in GEN APE with LoRA in the reported decoder ablation.
\end{abstract}

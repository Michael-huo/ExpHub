# VLMem-Codec IEEE Overleaf Project

This package is based on the IEEEtran journal template and contains the current manuscript draft for the VLMem-Codec paper.

Recommended Overleaf compiler: pdfLaTeX + BibTeX.

Main files:
- `main.tex`: manuscript source
- `references.bib`: BibTeX database
- `main.pdf`: local compile preview

The Methodology section is organized into three parts:
1. Framework Overview and Problem Formulation
2. Motion-Semantic Event Segmentation
3. Dual-Branch World-Model Decoding

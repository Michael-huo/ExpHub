# HVM-Codec IEEE Overleaf Project

This package is based on the IEEEtran journal template and contains the current manuscript draft with Introduction, Related Work, and the updated four-part Methodology section.

Recommended Overleaf compiler: pdfLaTeX + BibTeX.

Main files:
- `main.tex`: manuscript source
- `references.bib`: BibTeX database
- `main.pdf`: local compile preview

The Methodology section is organized into four parts:
1. Framework Overview and Problem Formulation
2. Motion-Semantic Event Segmentation for Boundary-Frame Selection
3. Controlled Language Memory Coding for Non-Transmitted Frames
4. World-Model Decoding and LoRA Fine-Tuning for VSLAM-Friendly Reconstruction

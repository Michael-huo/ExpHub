\section{Conclusion}
\label{sec:conclusion}
% ==================================================

This paper presented \HVM, a human-inspired image--text visual memory coding framework for edge--cloud VSLAM. The proposed formulation compresses dense image streams into boundary frames, motion memories, visual anchors, and compact language conditions, and reconstructs task-usable visual sequences on the cloud using an image--text conditioned world-model decoder with LoRA fine-tuning. Future experiments will validate the compression--trajectory trade-off across multiple VSLAM datasets and real robot scenes.

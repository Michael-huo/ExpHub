\subsubsection{Baseline Methods}
\label{subsubsec:baselines}
% TODO: Define all baselines precisely.
The following baselines are considered: raw frame transmission, standard image/video compression, uniform keyframes, boundary frames with a generic generator, boundary frames with language memory but without LoRA, and the full \HVM.

\subsubsection{Metrics}
\label{subsubsec:metrics}
% TODO: Define compression, VSLAM, runtime, and optional visual metrics.
Compression is measured by transmitted bytes and compression ratio. VSLAM quality is measured by APE, RPE, trajectory completeness, matched poses, and tracking success rate. Runtime is reported separately for edge encoding, cloud decoding, and VSLAM evaluation.
